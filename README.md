# PRODIGY_WD_04
A responsive Weather App built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

# Features
- Search weather by city name
- Real-time weather information
- Temperature, humidity, and wind speed
- Responsive UI

# Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
